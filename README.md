# The UX Review Blog

A responsive clone of the UX Review Blog landing page built using HTML5 and CSS3.

This project focuses on modern frontend development practices including Flexbox layouts, responsive design, semantic HTML, BEM naming conventions, and a CSS-only mobile navigation menu.

## Preview

![Project Preview](img/project-preview.png)

## Live Demo

https://the-ux-review-blog.vercel.app/

## Features

* Responsive design for mobile, tablet, and desktop screens
* CSS-only mobile navigation menu
* Flexbox-based layouts
* BEM naming convention
* Semantic HTML5 structure
* Modern blog landing page UI
* Google Fonts integration

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Google Fonts

## Learning Objectives

This project was built to practice and strengthen my understanding of:

* Semantic HTML structure
* CSS Positioning
* Flexbox layouts
* Responsive Web Design
* Media Queries
* BEM Methodology
* Mobile Navigation Patterns
* CSS-only Hamburger Menu using Checkbox and Label
* Layout recreation from a real-world design
